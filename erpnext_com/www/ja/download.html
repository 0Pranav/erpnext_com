インストール方法

イメージの利用にはOracle Virtual Boxが必要です。

  • https://www.virtualbox.org/

VirtualBoxに .ova ファイルをインポートします。 ほとんどのユーザーにとっては、初期設定で問題ありませんが、パフォーマンスに問題がある場合は変更が必要かもしれません。

ゲストOSには Debian GNU/Linux 7 を使用します。

イメージ内の初期アカウントは次の通りです：

  • username: erpnext
  • password: erpnext
  • mysql root password: erpnext
  • root password: erpnext
  • erpnext database name: erpnext
  • erpnext database password: erpnext

VM起動後、ホストOSのブラウザにて次のURLにアクセスします：

  • http://localhost:8080

ログイン情報は次の通りです：

  • user: Administrator
  • password: admin

イメージダウンロード

https://erpnext.com/downloads/ERPNext-100915.ova (1.1GB, MD5 Checksum - 753d07d58dfc03ba0d4793b768e69760)

作成日時：2015/9/10

イメージアップデート

VM内のERPNextをアップデートする場合、Virtual Boxのシェル (username: "erpnext", password: "erpnext") にログインし、次の通り実行します。 

cd frappe-bench
bench update

問題が発生した場合は、 https://discuss.erpnext.com にてIssueを登録してください。
